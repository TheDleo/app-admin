AppApi

This is the AppApi API

getContainers

List the names of the available containers in this storage. Use 'include_properties' to include any properties of the containers.

URL

https://yourdspname.cloud.dreamfactory.com/rest/app

HTTP Method

GET

Response Type

Containers

Parameters

    include_properties : boolean (query)
    optional

    Return all properties of the container, if any.

getContainer

Use 'include_properties' to get properties of the container. Use the 'include_folders' and/or 'include_files' to return a listing.

URL

https://yourdspname.cloud.dreamfactory.com/rest/app/{container}/

HTTP Method

GET

Response Type

FoldersAndFiles

Parameters

    container : string (path)

    The name of the container you want to retrieve the contents.

    include_properties : boolean (query)
    optional

    Return all properties of the container, if any.

    include_folders : boolean (query)
    optional

    Include folders in the returned listing.

    include_files : boolean (query)
    optional

    Include files in the returned listing.

    full_tree : boolean (query)
    optional

    List the contents of all sub-folders as well.

    zip : boolean (query)
    optional

    Return the zipped content of the folder.

getFile

By default, the file is streamed to the browser. Use the 'download' parameter to prompt for download. Use the 'include_properties' parameter (optionally add 'content' to include base64 content) to list properties of the file.

URL

https://yourdspname.cloud.dreamfactory.com/rest/app/{container}/{file_path}

HTTP Method

GET

Response Type

File

Parameters

    container : string (path)

    Name of the container where the file exists.

    file_path : string (path)

    Path and name of the file to retrieve.

    include_properties : boolean (query)
    optional

    Return properties of the file.

    content : boolean (query)
    optional

    Return the content as base64 of the file, only applies when 'include_properties' is true.

    download : boolean (query)
    optional

    Prompt the user to download the file from the browser.

URL

https://yourdspname.cloud.dreamfactory.com/rest/app/{container}/{folder_path}/

HTTP Method

GET

Response Type

FoldersAndFiles

Parameters

    container : string (path)

    The name of the container from which you want to retrieve contents.

    folder_path : string (path)

    The path of the folder you want to retrieve. This can be a sub-folder, with each level separated by a '/'

    include_properties : boolean (query)
    optional

    Return all properties of the folder, if any.

    include_folders : boolean (query)
    optional

    Include folders in the returned listing.

    include_files : boolean (query)
    optional

    Include files in the returned listing.

    full_tree : boolean (query)
    optional

    List the contents of all sub-folders as well.

    zip : boolean (query)
    optional

    Return the zipped content of the folder.
