Config

  • dsp_version : string
    Version of the DSP software.
  • db_version : string
    Version of the database schema.
  • allow_open_registration : boolean
    Allow guests to register for a user account.
  • open_reg_role_id : integer
    Default Role Id assigned to newly registered users.
  • allow_guest_user : boolean
    Allow app access for non-authenticated users.
  • guest_role_id : integer
    Role Id assigned for all guest sessions.
  • editable_profile_fields : string
    Comma-delimited list of fields the user is allowed to edit.
  • allowed_hosts : List[HostInfo]
    CORS whitelist of allowed remote hosts.
