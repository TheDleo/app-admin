FoldersAndFiles

  • name : string
    Identifier/Name for the current folder, localized to requested folder resource.
  • path : string
    Full path of the folder, from the service including container.
  • container : string
    Container for the current folder.
  • last_modified : string
    A GMT date timestamp of when the folder was last modified.
  • _property_ : string
    Storage type specific properties.
  • metadata : List[string]
    An array of name-value pairs.
  • folder : List[Folder]
    An array of contained folders.
  • file : List[File]
    An array of contained files.
