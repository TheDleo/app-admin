Welcome.

Choose a method on the left to begin exploring. This SDK or Seed App will grab all of the services available on your DSP, and give you access to them through a dom object (df currently in app.js).

If you still have the following lines in your /lib/sdk-init.js, you'll need to put in your dsp and app information : 
//replace this dsp_url with yours ( leave the /rest/api_docs part )
var dsp_url = "https://_your_dsp_hostname_here_/rest/api_docs";
//replace this app_name with yours
var app_name = "_your_app_api_name_here_";
Lets look at Authentication and CRUD examples to get you started.

Authentication

Authenticating against your DSP requires you to post to /user/session. The SDK has a user.login convenience method to handle this for you.

  var body = {
  email: "dspuser@host.com",
  password: "password"
  };
  window.df.apis.user.login({body: body},
  function (response) {
     window.authorizations.add("X-DreamFactory-Session-Token",
     new ApiKeyAuthorization("X-Dreamfactory-Session-Token",
     response.body.data.session_id, 'header'));
  });
Notice where we add the new X-DreamFactory-Session-Token. This will keep your user logged in for the duration of their session.

To log a user out, we've provided a convenience method for that as well.

window.df.apis.user.logout()
